# Hello Health App
## Beskrivelse

Er en python app som ønsker brukere velkommmen ved hjelp av en venv.
Formålet er å vise at Python er riktig installert, at virtuelt miljø fungerer, og at Git kan brukes til å lagre og dele kode 

## Installasjonsinstruksjoner

### Nummererte, trinnvise instruksjoner som dekker:

Cloning the repository
1. Gå inn på Github 
2. Clone fra HTTP eller SSH- kopier linken som stå der 
3. Gå inn på gå inn på VSCOD og lim inn linken i terminalen med med koden Git clone også linken etter 

Å lage et virtuelt miljø
1. Åpne terminalen 
2. Skriv inn (på mac) python3 -m venv venv


Aktivering av det virtuelle miljøet
1. For å aktiverer så bruker man koden source venv/bin/activate 

Kjører applikasjonen
1. For å åpne i VSCODE så åpner du terminalen og skriver code .
